1. 프로젝트 개요

MyMoney는 가정 재무 현황을 Excel 파일로 관리하는 사용자를 위한 웹 시각화 앱이다. 뱅크샐러드 내보내기 형식의 Excel 파일을 업로드하면 고객정보·현금흐름·고정비·재무목표·부동산 분석 데이터를 자동 추출하여 차트와 대시보드로 제공한다.

영역기술
FrontendReact 19, TypeScript, Vite, React Router 7, React Query 5, Recharts, Tailwind CSS 4
BackendFastAPI, SQLAlchemy 2, Pydantic 2, Alembic, APScheduler
DatabasePostgreSQL
인프라Docker Compose

3. 요구사항 정의

3.1 기능 요구사항 (FR)

FR-001 ✅ 구현 완료 Excel 파일 업로드

설명: 사용자가 가계부 Excel 파일을 웹 UI를 통해 업로드할 수 있다.

처리: 모든 시트를 파싱 → DataRecord JSON 저장 → UploadHistory, SheetData 레코드 생성

제약: .xlsx/.xls만 허용, 파일 크기 제한 필요 (NFR-002)

FR-002 ✅ 구현 완료 고객 정보 추출·관리

설명: 뱅샐현황 시트의 고객정보를 추출하여 조회할 수 있다.

처리: "이름" 헤더 탐색 → 이름·성별·나이·신용점수·이메일 추출 → 동일 이름 시 파일 날짜 기반 최신 우선

출력: Customer[]

FR-003 ✅ 구현 완료 현금흐름 분석

설명: 수입/지출 항목별 월별 현금흐름을 추출하여 조회·시각화한다.

처리: "2.현금흐름현황" 섹션 탐색 → 항목별 총계·월평균·월별 13개월 추출 → 복수 파일 월별 병합

출력: CashFlow[], 월별 트렌드 차트

FR-004 🔲 신규 고정비 관리

설명: 고정비 시트의 계좌·은행·항목별 월별 고정지출을 추출하여 조회·분석한다.

처리: "계좌" 헤더 탐색 → 성향 기반 카테고리 분류 (저축/용돈/보험/교육) → 월별 JSON 저장

출력: FixedExpense[], 카테고리별 파이 차트

원본 시트: 고정비 (26)

FR-005 🔲 신규 월별 재무 요약

설명: 총 결산 시트의 월별 수입·지출·순수익·누적 데이터를 추출하여 연간 대시보드로 제공한다.

처리: 행 레이블 매핑 → 연도·월별 6개 지표 추출

출력: MonthlySummary[], 연간 막대 차트, 누적 순수익 라인 차트

원본 시트: 총 결산

FR-006 🔲 신규 재무 목표 관리

설명: 분양금 계획 시트의 목표금액·기간·진행률 및 계획 vs 실제 집행을 추적한다.

처리: 메타데이터 영역 추출 → 계획/집행 섹션을 각각 JSON 배열로 저장

출력: FinancialGoal, 진행률 게이지, 계획 vs 집행 비교 차트

원본 시트: 평촌 자이 퍼스티니 분양금 계획

FR-007 🔲 신규 부동산 투자 분석

설명: 부동산 수익분석 시트의 자산 구조·ROE·레버리지 지표를 추출하여 투자 성과 요약을 제공한다.

처리: 취득원가·자기자본·타인자본·시세차익·ROE·레버리지 배수·가속계수 추출

출력: RealEstateAnalysis, 자산 구조 파이 차트, 레버리지 비교 테이블

원본 시트: 부동산 수익분석, Sheet3

FR-008 🔶 확장 데이터 시각화

현재 구현: 현금흐름 월별 트렌드, 차트 클릭 세부 내역 팝업 모달

신규 추가: 고정비 카테고리 파이 차트 (FR-004) / 월별 순수익 누적 차트 (FR-005) / 재무 목표 진행률 게이지 (FR-006) / 투자 수익률 요약 카드 (FR-007)

# 1단계: 파일 상단에 상수 정의
NEW_COL_FIELD1 = "1"
NEW_MONTHLY_COL_RANGE = range(7, 19)   # 01~12월
NEW_EXCLUDE_KEYWORDS = ["헤더명", "합계", ...]

# 2단계: 추출 함수
def extract_and_save_new_model(db, sheet_id, upload_id):
    records = db.query(DataRecord).filter(
        DataRecord.sheet_id == sheet_id
    ).order_by(DataRecord.row_index).all()

    # 헤더 행 탐색
    header_row = next(
        (r.row_index for r in records
         if _to_str(r.data.get("1")) == "헤더키워드"), None
    )

    results = []
    for record in records:
        if record.row_index <= header_row:
            continue
        item_name = _to_str(record.data.get(NEW_COL_FIELD1))
        if not item_name or any(k in item_name for k in NEW_EXCLUDE_KEYWORDS):
            continue

        # 월별 데이터 추출
        monthly_data = {
            f"2026-{i - 6:02d}": _to_float(record.data.get(str(i)))
            for i in NEW_MONTHLY_COL_RANGE
            if _to_float(record.data.get(str(i))) is not None
        }

        # Upsert (같은 sheet_id + item_name)
        existing = db.query(NewModel).filter(
            NewModel.sheet_id == sheet_id,
            NewModel.item_name == item_name,
            NewModel.upload_id == upload_id
        ).first()

        if existing:
            existing.monthly_data = monthly_data
            existing.updated_at = datetime.now()
        else:
            existing = NewModel(sheet_id=sheet_id, item_name=item_name,
                               monthly_data=monthly_data, upload_id=upload_id,
                               data_record_id=record.id)
            db.add(existing)
        results.append(existing)

    db.commit()                       # ← 루프 밖에서 단일 commit (N+1 방지)
    for r in results:
        db.refresh(r)
    return results
